App (최상위 컴포넌트)

전체 앱의 상태 관리
할 일 목록 데이터 보관
다른 모든 컴포넌트들을 감싸는 컨테이너 역할


TodoBoard (메인 보드)

전체 할 일 관리 영역
TodoInput과 TodoList를 포함
필터링이나 정렬 기능 포함 가능


TodoInput (입력 컴포넌트)

새로운 할 일을 입력하는 폼
제출 버튼 포함
입력된 데이터를 상위 컴포넌트로 전달


TodoList (목록 컴포넌트)

여러 개의 TodoItem을 포함
전체 목록을 관리하고 표시


TodoItem (개별 할 일 항목)

각각의 할 일 표시
체크박스로 완료 여부 표시
삭제 버튼
수정 기능
